# 📌 Overview

## 🎯 Objetivo

Este projeto tem como objetivo demonstrar a construção de uma API utilizando **FastAPI** com autenticação baseada em **Bearer Token (JWT)**, incluindo:

* Geração de token de acesso
* Proteção de endpoints
* Simulação de recebimento de follow-up

---

## 🧩 Contexto

A API simula um cenário onde sistemas externos enviam **follow-ups de clientes**, sendo necessário:

* Garantir autenticação
* Validar requisições
* Manter rastreabilidade

---

## 🏗️ Escopo

### Inclui

* Endpoint de autenticação
* Endpoint protegido
* Estrutura modular
* Documentação técnica

### Não inclui (neste momento)

* Persistência em banco
* Gestão de usuários real
* Refresh token
* Autorização por roles

---

## 👥 Usuários

* Sistemas externos (integrações)
* Desenvolvedores (uso técnico)
